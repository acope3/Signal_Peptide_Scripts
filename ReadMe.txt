Data contains the sequences and gene expression data used in this project.
Results contains the completed model fittings described in the main text of the manuscript.
Note that some model output might differ from others due to changes in the AnaCoDa software package.
These changes to AnaCoDa only impacted the format of the model output, not the underlying model.
